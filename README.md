# 🌾 AgroTech - Inovação e Tecnologia no Campo

Um site moderno e responsivo sobre inovação tecnológica na agricultura, desenvolvido com HTML5, CSS3 e JavaScript vanilla.

## ✨ Características

- ✅ **Design Responsivo** - Funciona perfeitamente em mobile, tablet e desktop
- ✅ **Menu Mobile** - Hamburger menu intuitivo para dispositivos menores
- ✅ **Performance Otimizada** - CSS com variáveis customizadas e estrutura semântica
- ✅ **Acessibilidade** - HTML semântico e navegação por teclado
- ✅ **Animações Suaves** - Efeitos visuais elegantes com Intersection Observer
- ✅ **SEO Amigável** - Meta tags e estrutura otimizada para buscadores
- ✅ **Contadores Animados** - Estatísticas com animação ao scroll
- ✅ **Zero Dependências** - Apenas HTML, CSS e JavaScript puro

## 📁 Estrutura do Projeto

```
Agrinho-2026-/
```

## 🚀 Como Usar

1. **Clone ou baixe o repositório**
   ```bash
   git clone https://github.com/pereirasilvalucas2212-beep/Agrinho-2026-.git
   ```

2. **Abra o arquivo `index.html` em seu navegador**
   ```bash
   # Opção 1: Abrir direto no navegador
   open index.html
   
   # Opção 2: Usar um servidor local (Python)
   python -m http.server 8000
   
   # Opção 3: Usar Live Server (VS Code)
   # Extensão: Live Server
   ```

3. **Customize conforme necessário**
   - Edite as cores em `:root` no `styles.css`
   - Adicione seu próprio conteúdo no `index.html`
   - Estenda as funcionalidades no `script.js`

## 🎨 Personalização

### Alterar Cores
Edite as variáveis CSS em `styles.css`:

```css
:root {
    --primary-color: #2d5016;      /* Verde escuro */
    --secondary-color: #52b788;    /* Verde claro */
    --accent-color: #ffb703;       /* Amarelo */
}
```

### Adicionar Novas Seções
1. Adicione um novo `<section>` com um `id` único
2. Crie estilos CSS para essa seção
3. Adicione um link de navegação no `<header>`

### Modificar Conteúdo
Todos os textos podem ser alterados diretamente no `index.html`. A estrutura está bem organizada para fácil manutenção.

## 📱 Responsividade

O site é totalmente responsivo com breakpoints em:
- **Desktop**: Acima de 1200px
- **Tablet**: 768px a 1199px
- **Mobile**: Abaixo de 767px
- **Pequenos Mobile**: Abaixo de 480px

## 🎯 Seções do Site

1. **Header/Navbar** - Navegação sticky com menu mobile
2. **Hero Section** - Apresentação principal com CTA
3. **Technologies** - Grid de 3 cards com funcionalidades
4. **Impact** - Estatísticas com animação de contadores
5. **CTA** - Call-to-Action secundário
6. **Footer** - Informações de contato e links

## 🔧 Funcionalidades JavaScript

- **Menu Mobile Toggle** - Abre/fecha menu em dispositivos pequenos
- **Smooth Scroll** - Scroll suave para seções
- **Intersection Observer** - Anima elementos ao entrar na viewport
- **Active Link Highlighting** - Destaca link de navegação atual
- **Counter Animation** - Anima números nas estatísticas
- **Keyboard Navigation** - Suporte a navegação com teclado
